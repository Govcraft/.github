# Govcraft Community Health Files

This repo contains default community health files for all Govcraft repositories.

## What's Here

- **FUNDING.yml** - Sponsorship links that appear on all Govcraft repos
- **profile/README.md** - The organization profile you see at [github.com/Govcraft](https://github.com/Govcraft)

## About Govcraft

Govcraft is a one-person operation run by [@rrrodzilla](https://github.com/rrrodzilla). If you find any of these projects useful, consider [becoming a sponsor](https://github.com/sponsors/Govcraft).
